331 *

333 * [current_lo, current_hi, status_word, reserved].

344 uint16_t status_word; ///< Raw status word returned by the PLC for callers that need bit-exact inspection.

507 * @class SlmpClient
508 * @brief Main SLMP client implementation.
509 *
510 * This class is the core low-level client. It is designed to be:
511 * - **Memory Efficient**: No dynamic allocation. User provides buffers.

517 * optional helper facade in @ref slmp_high_level.h.

529 * @code
